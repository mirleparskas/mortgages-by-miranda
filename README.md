# Mortgages by Miranda

Static website for an Ontario mortgage broker brand, built with plain HTML and CSS.

## Files

- `index.html` - main website page with visible FAQ content and JSON-LD structured data.
- `styles.css` - responsive visual styling.
- `assets/home-financing-visual.svg` - local visual asset.
- `llms.txt` - concise AI crawler context.
- `robots.txt` - crawler instructions.
- `sitemap.xml` - XML sitemap.

## Before Publishing

Replace these placeholders:

- `https://mortgagesbymiranda.ca/` if the final domain is different.
- `hello@mortgagesbymiranda.ca`
- `416-555-0142`
- Brokerage name, license details, legal disclaimers, privacy policy link, and any required provincial compliance language.

Add real proof only when available:

- Verified reviews
- Awards
- Lender access claims
- Current rates
- Photos of Miranda or the office
