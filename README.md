# Mortgages by Miranda

Static website for an Ontario mortgage broker brand, built with plain HTML and CSS.

## Files

- `index.html` - main website page with visible FAQ content and JSON-LD structured data.
- `styles.css` - responsive visual styling.
- `assets/home-financing-visual.svg` - local visual asset.
- `llms.txt` - concise AI crawler context.
- `robots.txt` - crawler instructions.
- `sitemap.xml` - XML sitemap.
